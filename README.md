## Description

Nordstrom rack website
https://www.nordstromrack.com/

## Setup/Installation Requirements
1. Clone the following repository: https://github.com/MelYasuda/nordstrom-rack
2. On terminal, go to nordstrom-rack directory
3. Run "npm install"
4. Run "ng serve -o"
